Instructions for Setup: 

1) Rename the file to something less suspicous with the .sh suffix
2) Use "chmod +x "filename" to make the file an executable
3) Input the file locaiton into the SCRIPT_PATH variable 
4) (Optional) Alter the name of each fake cronjob to fit your needs
6) Run the command in the files directory
7) Use crontab -r to clear the script from running.
